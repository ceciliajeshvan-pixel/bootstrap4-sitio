# Bootstrap 4 Website - TechAgency

Sitio web maquetado con Bootstrap 4, ScrollReveal y JavaScript vanilla.

## Preview

Sitio publicado en GitHub Pages: `https://TU-USUARIO.github.io/TU-REPO/`

## Tecnologías usadas

- Bootstrap 4.5
- Font Awesome 5
- ScrollReveal.js
- JavaScript vanilla (sin jQuery para la lógica)

## Estructura

```
bootstrap4-sitio/
├── src/
│   ├── index.html        ← código fuente
│   ├── css/
│   │   └── styles.css
│   └── js/
│       └── main.js
├── docs/
│   ├── index.html        ← copia para GitHub Pages
│   ├── css/
│   │   └── styles.css
│   └── js/
│       └── main.js
└── README.md
```

## Secciones

- Navbar fija con efecto scroll
- Hero con animación
- Servicios (6 cards)
- Stats con contador animado
- Portfolio (6 proyectos)
- Equipo (3 integrantes)
- Formulario de contacto
- Footer

## Cómo subir a GitHub Pages

1. Subir el repositorio a GitHub
2. Ir a **Settings → Pages**
3. Source: rama `main`, carpeta `/docs`
4. Guardar y esperar unos minutos
